## 3. 使用哪些函數（由程式碼拆解）
本作品在 Desmos 中使用「參數曲線」為主，函數類型如下：

1. 直線與限制條件（邊界）
   - `y=0 {0<x<480}`
   - `y=360 {0<x<480}`
   - `x=0 {0<y<360}`
   - `x=480 {0<y<360}`
   這些是一次函數與不等式限制，形成影格邊界。

2. 線段（角落輪廓）
   - 程式在 `backend.py` 中以線性插值表示線段：
   - 參數式：
     ```
     (x(t), y(t)) = ((1-t)x0 + t x1, (1-t)y0 + t y1), 0<=t<=1
     ```

3. 三次貝賽爾曲線（平滑輪廓）
   - 程式在 `backend.py` 內將輪廓輸出成三次貝賽爾曲線參數式。
   - 標準公式：
     ```
     B(t) = (1-t)^3 P0 + 3(1-t)^2 t P1 + 3(1-t) t^2 P2 + t^3 P3, 0<=t<=1
     ```
   - 每一條曲線對應 Desmos 上的一個參數曲線。

4. 影格控制變數
   - `frame` 變數用來切換不同影格，形成動畫。

## 4. 標示圖形中的函數（Desmos 顯示）
因單幀曲線數量很大，若全部標示會嚴重遮擋圖形，因此採用「取樣顯示」：

- 邊界直線全部顯示標籤（`y=0`, `y=360`, `x=0`, `x=480`）。
- 每一幀只標示前 10 條曲線，避免畫面過度擁擠。
- 標籤內容為曲線的參數式（由 `backend.py` 輸出）。

實作位置：`index.html` 內對 Desmos expression 加上 `label` 與 `showLabel`。

## 5. 原理簡述（微積分角度）
影像輪廓可視為由大量平滑曲線拼成。  
三次貝賽爾曲線是一種參數曲線，能用少量控制點表達彎曲程度。  
當影格輪廓被拆成許多小段貝賽爾曲線後，只要逐影格更新這些曲線，就能形成連續動畫。  
因此「用參數曲線近似輪廓」與「影格快速播放」就是本作品的數學核心。
